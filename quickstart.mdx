description: "Install MUI X and render a fully functional Data Grid in your React app in a few steps, with TypeScript types and pagination support included from the start."

This guide walks you through installing MUI X and rendering your first component. You'll install the Community Data Grid package, set up the required peer dependencies, and have a working table on screen in minutes. The same pattern applies to all other MUI X component families.

<Steps>
  <Step title="Install dependencies">
    Install the Community Data Grid package along with its peer dependencies. MUI X requires React 17, 18, or 19, Material UI, and the Emotion styling engine.

    <CodeGroup>

    ```bash npm
    npm install @mui/x-data-grid @mui/material @emotion/react @emotion/styled
    ```

    ```bash yarn
    yarn add @mui/x-data-grid @mui/material @emotion/react @emotion/styled
    ```

    ```bash pnpm
    pnpm add @mui/x-data-grid @mui/material @emotion/react @emotion/styled
    ```

    </CodeGroup>
  </Step>

  <Step title="Render your first Data Grid">
    Import `DataGrid` from `@mui/x-data-grid` and provide it with `rows` and `columns` props. Each row must have a unique `id` field, and each column definition requires a `field` and `headerName`.

    ```tsx
    import * as React from 'react';
    import { DataGrid } from '@mui/x-data-grid';

    const columns = [
      { field: 'id', headerName: 'ID', width: 70 },
      { field: 'firstName', headerName: 'First name', width: 130 },
      { field: 'lastName', headerName: 'Last name', width: 130 },
      { field: 'age', headerName: 'Age', type: 'number', width: 90 },
    ];

    const rows = [
      { id: 1, lastName: 'Snow', firstName: 'Jon', age: 14 },
      { id: 2, lastName: 'Lannister', firstName: 'Cersei', age: 31 },
      { id: 3, lastName: 'Lannister', firstName: 'Jaime', age: 31 },
      { id: 4, lastName: 'Stark', firstName: 'Arya', age: 11 },
      { id: 5, lastName: 'Targaryen', firstName: 'Daenerys', age: null },
    ];

    export default function BasicDataGrid() {
      return (
        <div style={{ height: 400, width: '100%' }}>
          <DataGrid rows={rows} columns={columns} />
        </div>
      );
    }
    ```

    <Note>
      The `DataGrid` component requires a parent element with a defined height. Without it, the grid will not render correctly.
    </Note>
  </Step>

  <Step title="Add TypeScript types">
    MUI X ships full TypeScript definitions. Use `GridColDef` and `GridRowsProp` to type your columns and rows for better editor support and type safety.

    ```tsx
    import * as React from 'react';
    import { DataGrid, GridColDef, GridRowsProp } from '@mui/x-data-grid';

    const columns: GridColDef[] = [
      { field: 'id', headerName: 'ID', width: 70 },
      { field: 'firstName', headerName: 'First name', width: 130 },
      { field: 'lastName', headerName: 'Last name', width: 130 },
      { field: 'age', headerName: 'Age', type: 'number', width: 90 },
    ];

    const rows: GridRowsProp = [
      { id: 1, lastName: 'Snow', firstName: 'Jon', age: 14 },
      { id: 2, lastName: 'Lannister', firstName: 'Cersei', age: 31 },
      { id: 3, lastName: 'Lannister', firstName: 'Jaime', age: 31 },
      { id: 4, lastName: 'Stark', firstName: 'Arya', age: 11 },
    ];

    export default function TypedDataGrid() {
      return (
        <div style={{ height: 400, width: '100%' }}>
          <DataGrid rows={rows} columns={columns} />
        </div>
      );
    }
    ```
  </Step>

  <Step title="Enable built-in features">
    Pass additional props to the `DataGrid` to turn on built-in features like checkbox selection, pagination page size options, and row selection behavior.

    ```tsx
    <DataGrid
      rows={rows}
      columns={columns}
      initialState={{
        pagination: {
          paginationModel: { pageSize: 5, page: 0 },
        },
      }}
      pageSizeOptions={[5, 10, 25]}
      checkboxSelection
      disableRowSelectionOnClick
    />
    ```
  </Step>
</Steps>
