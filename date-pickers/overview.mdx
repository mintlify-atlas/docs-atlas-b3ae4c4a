MUI X Date & Time Pickers is a collection of React components that let users select dates, times, and date ranges. The components adapt automatically to desktop and mobile interactions, integrate with your existing MUI theme, and support multiple date libraries through a pluggable adapter system.

## Available components

### Community package (`@mui/x-date-pickers`)

The Community package includes all single-value pickers and their static and responsive variants:

- **DatePicker** — select a single date from a calendar popover
- **TimePicker** — select a time using a clock or digital clock UI
- **DateTimePicker** — select both date and time in one picker
- **StaticDatePicker / StaticTimePicker / StaticDateTimePicker** — inline (always-visible) variants
- **DesktopDatePicker / MobileDatePicker** — force a specific presentation mode
- **DigitalClock / MultiSectionDigitalClock** — standalone time selection UIs

### Pro package (`@mui/x-date-pickers-pro`)

The Pro package adds range-selection pickers that require a [commercial license](/licensing):

- **DateRangePicker** — select a start and end date across one or more calendar months
- **DateTimeRangePicker** — select start and end date-times
- **TimeRangePicker** — select a time range

## Adapter system

Every picker requires a date library adapter passed to `LocalizationProvider`. The adapter translates between picker internals and your chosen date library. Four adapters ship with the packages:

| Adapter | Date library |
|---|---|
| `AdapterDayjs` | [Day.js](https://day.js.org) |
| `AdapterLuxon` | [Luxon](https://moment.github.io/luxon) |
| `AdapterMoment` | [Moment.js](https://momentjs.com) |
| `AdapterDateFns` | [date-fns](https://date-fns.org) |

See [localization](/date-pickers/localization) for setup instructions and language locale support.
