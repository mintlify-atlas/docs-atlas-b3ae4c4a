The MUI X Data Grid is a powerful React component for displaying and interacting with tabular data. It handles sorting, filtering, pagination, editing, and more out of the box, and scales from simple read-only tables to complex enterprise data management interfaces.

## Tiers

The Data Grid is available in three tiers. Each tier is a separate npm package.

| Tier | Package | License |
|------|---------|---------|
| Community | `@mui/x-data-grid` | MIT |
| Pro | `@mui/x-data-grid-pro` | Commercial |
| Premium | `@mui/x-data-grid-premium` | Commercial |

<Note>
Pro and Premium require a valid license key. See the [licensing](/licensing) page for details.
</Note>

## Key features

**Community (free)**

- Column sorting, filtering, and resizing
- Client-side pagination
- Row selection (single and multiple)
- Cell and row editing
- CSV export
- Virtualized rendering for large datasets
- Column visibility toggling
- Density controls
- Accessible by default

**Pro (commercial)**

Everything in Community, plus:

- Column pinning (freeze columns to the left or right)
- Column reordering via drag and drop
- Row pinning
- Row reordering
- Multi-column sorting
- Multi-column filtering
- Tree data and master-detail panels
- Server-side data source API
- Lazy loading and infinite scroll

**Premium (commercial)**

Everything in Pro, plus:

- Row grouping
- Aggregation (sum, avg, min, max, count)
- Excel export
- Clipboard import/paste
- Pivoting (experimental)
