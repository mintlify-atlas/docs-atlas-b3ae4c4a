MUI X is a suite of advanced React UI components built for complex, data-rich applications. It extends the foundation of Material UI with components that go beyond what the open-source community can reasonably maintain alone—covering data grids, date and time pickers, charts, and tree views. MUI X is open-core: the Community edition is MIT-licensed and free forever, while the Pro and Premium tiers unlock additional features under a commercial license.

## Component families

MUI X ships four major component families, each available across Community, Pro, and Premium tiers.

## Licensing tiers

MUI X uses an open-core model with three tiers. All MIT-licensed code remains MIT-licensed forever.

| Tier | License | Cost |
|------|---------|------|
| Community | MIT | Free |
| Pro | Commercial | Paid |
| Premium | Commercial | Paid |

The **Community** tier includes the core components: `@mui/x-data-grid`, `@mui/x-date-pickers`, `@mui/x-charts`, and `@mui/x-tree-view`. These are free to use in any project.

The **Pro** tier adds advanced features such as multi-filtering, multi-sorting, column resizing, column pinning for the Data Grid; date range pickers; advanced charts; and drag-and-drop Tree View reordering.

The **Premium** tier unlocks the most advanced Data Grid capabilities, including row grouping and Excel exporting, plus everything in Pro.

See the [licensing page](/licensing) for a full breakdown of which packages belong to each tier and how to apply a license key.

## Design philosophy

MUI X components can be used standalone or integrated with Material UI. They follow the same theming and customization API as Material UI, but they are not locked to it—you can adapt them to any design system. The library is written in TypeScript, ships full type definitions, and is tested for accessibility.
