MUI X Charts is a React charting library built on top of Material UI. It ships composable SVG components for every common chart type, an integrated tooltip and legend system, and full TypeScript support. The core chart components are available under the MIT Community license. Pro and Premium tiers add advanced chart types such as heatmaps, funnel charts, candlestick charts, and zoom/pan interactions.

## Chart types at a glance

| Component | What it renders |
|---|---|
| `BarChart` | Vertical or horizontal bar series, supports stacking |
| `LineChart` | Line and area series, marks, curve interpolation |
| `ScatterChart` | X/Y point clouds with optional Z-axis sizing |
| `SparkLineChart` | Compact inline line or bar chart, no axes |
| `PieChart` | Arc-based proportional charts, supports donut variant |
| `RadarChart` | Polygon-based multivariate comparison charts |
| `Gauge` | Arc-shaped progress/value indicator |

## Composition approach

Every chart component is a convenience wrapper around a set of lower-level building blocks. You can use the wrappers for the common case or compose the primitives directly when you need full control:

- **`ChartsDataProvider`** — context that provides scales, series, and axis state
- **`ChartsSurface`** — SVG viewport with correct dimensions and margins
- **`BarPlot` / `LinePlot` / `PiePlot` / …** — series renderers
- **`ChartsAxis`** / **`ChartsXAxis`** / **`ChartsYAxis`** — axis components
- **`ChartsLegend`** — legend renderer
- **`ChartsTooltip`** — tooltip overlay
- **`ChartsGrid`** — background grid lines

<Note>
All chart components accept a `slots` and `slotProps` prop that lets you replace any internal sub-component without rewriting the entire chart.
</Note>

## Licensing tiers

| Tier | Package | Notable additions |
|------|---------|-------------------|
| Community (MIT) | `@mui/x-charts` | All core chart types (bar, line, scatter, pie, radar, gauge, sparkline) |
| Pro (commercial) | `@mui/x-charts-pro` | Zoom/pan, brush slider, heatmap, funnel chart, sankey |
| Premium (commercial) | `@mui/x-charts-premium` | Candlestick chart, WebGL layer, all Pro features |

All packages require a license key for the Pro and Premium tiers. See the [licensing page](/licensing) for setup instructions.
