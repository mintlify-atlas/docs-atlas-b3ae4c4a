## Get started

<Steps>
  <Step title="Install a package">
    Choose the component suite you need and install it via npm, yarn, or pnpm.
    ```bash
    npm install @mui/x-data-grid @mui/material @emotion/react @emotion/styled
    ```
  </Step>
  <Step title="Import and use the component">
    Import the component and add it to your React app with minimal configuration.
    ```tsx
    import { DataGrid } from '@mui/x-data-grid';

    const columns = [
      { field: 'id', headerName: 'ID', width: 90 },
      { field: 'name', headerName: 'Name', width: 150 },
    ];

    const rows = [{ id: 1, name: 'Alice' }, { id: 2, name: 'Bob' }];

    export default function App() {
      return <DataGrid rows={rows} columns={columns} />;
    }
    ```
  </Step>
  <Step title="Explore advanced features">
    Browse the component documentation to add filtering, sorting, theming, and more.
  </Step>
</Steps>

<Note>
